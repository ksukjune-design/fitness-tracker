# 팀 운동 현황 관리 애플리케이션

조직원들의 운동 현황을 기록하고 관리할 수 있는 웹 애플리케이션입니다.

## 주요 기능

### 1. 팀원 관리
- 팀원 추가, 수정, 삭제
- 기초 신체정보 수집 (키, 몸무게, 나이, 성별, 체지방률, 근육량)
- 운동 목적 선택 (체중 감량, 근육 증가, 지구력 향상, 유연성 향상, 근력 강화, 일반 건강)

### 2. 맞춤형 운동 계획
- 팀원별 개인화된 운동 계획 설정
- 다양한 운동 종류 추가 (유산소, 근력, 유연성, 균형)
- 주간 스케줄 관리 (요일별 운동 배정)

### 3. 운동 기록
- 일별 운동 기록
- 운동 완료 여부 추적
- 운동 시간, 세트, 횟수, 무게 기록
- 기분 상태 기록
- 메모 기능

### 4. 통계 및 모니터링
- 일간/주간/월간 통계
- 운동 시간 추이 차트
- 운동 횟수 차트
- 총 운동 횟수, 총 운동 시간, 평균 운동 시간

### 5. 팀 진척도 공유
- 모든 팀원의 운동 현황 한눈에 보기
- 최근 7일 운동 현황
- 주간 목표 달성률
- 팀원 간 격려 메시지 전송 및 확인

## 기술 스택

- **React 18** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Vite** - 빌드 도구
- **React Router** - 라우팅
- **Recharts** - 차트 라이브러리
- **date-fns** - 날짜 처리
- **lucide-react** - 아이콘
- **LocalStorage** - 데이터 저장

## 사용 방법

1. **팀원 추가**: 대시보드 또는 팀원 관리 페이지에서 "팀원 추가" 버튼 클릭
2. **신체정보 입력**: 팀원 추가 시 기초 신체정보와 운동 목적 선택
3. **운동 계획 설정**: 팀원 상세 페이지에서 "운동 계획 만들기" 클릭
4. **운동 기록**: 팀원 상세 페이지에서 "운동 기록하기" 클릭하여 일별 운동 기록
5. **통계 확인**: 팀원 상세 페이지에서 "통계 보기" 클릭하여 일/주/월간 통계 확인
6. **팀 진척도 확인**: 상단 네비게이션에서 "팀 진척도" 클릭하여 모든 팀원의 현황 확인 및 격려 메시지 전송

## 데이터 저장

모든 데이터는 브라우저의 LocalStorage에 저장됩니다. 브라우저를 닫아도 데이터가 유지됩니다.

## AWS 배포

AWS에 배포하는 방법은 [AWS_DEPLOYMENT.md](./AWS_DEPLOYMENT.md) 또는 [QUICK_START_AWS.md](./QUICK_START_AWS.md)를 참고하세요.

2. AWS Amplify 콘솔에서 앱 생성

### S3 + CloudFront 배포
```bash

# 빌드 및 배포
npm run build
npm run deploy:s3
```
